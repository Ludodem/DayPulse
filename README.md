# DayPulse

**"Prends le pouls de ta journée"**

A simple, responsive web app to rate each day across customizable metrics (1-10) and visualize your trends over time.

**Live:** https://ludodem.github.io/DayPulse/

## Features

- **Daily scoring** — Rate your day on multiple metrics (Energy, Happiness, Focus, or your own)
- **Calendar heatmap** — Monthly and yearly views with color-coded days (red to green)
- **Statistics** — Weekly and monthly bar charts with metric filtering
- **Responsive** — Mobile-first design with desktop layout
- **Offline** — All data stored in localStorage, no account needed
- **Export/Import** — Backup and restore your data as JSON

## Tech stack

React + TypeScript + Vite + Tailwind CSS + Recharts

## Development

```bash
npm install
npm run dev
```

## Deployment

Automatically deployed to GitHub Pages on every push to `master` via GitHub Actions.
